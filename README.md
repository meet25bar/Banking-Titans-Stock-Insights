# Banking-Titans-Stock-Insights

🏦 Banking Titans: Stock Insights

A data-driven analysis of major U.S. bank stocks using Python to uncover trends, compare performance, and visualize financial market behavior.

📊 Project Overview

This project analyzes historical stock data of leading banking institutions:

Bank of America (BAC)
Citigroup (C)
Goldman Sachs (GS)
JPMorgan Chase (JPM)
Morgan Stanley (MS)
Wells Fargo (WFC)

The goal is to explore stock performance, identify trends, and gain insights using data visualization and statistical techniques.

🚀 Features
-📈 Stock price visualization
-📉 Moving averages (trend analysis)
-🔍 Comparative analysis of multiple banks
-🕯️ Candlestick charts for detailed insights
-📊 Distribution plots of returns

🛠️ Tech Stack
-Python
-Pandas
-NumPy
-Matplotlib
-Seaborn
-yFinance
-Plotly / Cufflinks
